# qplayer
---
## 介绍
这是一款简单的音乐播放器，基于 Qt C++ 开发，开发环境为 QtCreator, 音乐播放器界面 UI 设计较为简陋。

## 功能
* 目前支持常见格式的音频文件的播放(\*.mp3, \*.wav, \*.wma等)
* 加载本地音乐文件
* 播放/暂停
* 调节音乐播放音量
* 调节音乐播放进度
* 切换曲目上一首和下一首
* 四种播放模式: 单曲播放，列表播放，单曲循环，列表循环
* 显示歌词(目前只支持 .lrc 格式的歌词文件)
